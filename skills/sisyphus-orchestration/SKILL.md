# Sisyphus Orchestration

## When to spawn

| Need | Agent |
|------|--------|
| Where is X? | explore |
| Design/tradeoffs | oracle |
| Docs/API research | librarian |
| Deep multi-file implement | hephaestus |
| Interview plan | prometheus (`/plan`) |
| Execute approved plan | atlas (`/start-work`) |

## Protocol

1. State the goal in one sentence.
2. Spawn specialist with a crisp brief.
3. Integrate results; update todos.
4. Verify; only then mark complete or `<promise>DONE</promise>` in loops.
